# Desafio Smart NX

Aplicacao React desenvolvida para o desafio tecnico da Smart NX, consumindo a SWAPI para listar personagens de Star Wars.

## Funcionalidades

- Listagem de personagens consumindo a API SWAPI
- Exibicao em grid com cards
- Paginacao com 10 registros por pagina
- Navegacao entre paginas
- Filtro por nome
- Traducao de campos da API para portugues

## Deploy no Vercel

[Clique aqui](https://desafio-smart-nx.vercel.app)

## Instalando o Desafio

Siga os passos abaixo:

### Clone este repositorio

```bash
git clone https://github.com/jefranca/desafio-smart-nx.git
```

### Instale as dependencias

```bash
npm i
```

### Inicie o projeto

```bash
npm start
```
